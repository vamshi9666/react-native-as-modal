# react-native-rn-as-modal

HOC to convert a component into modal

## Installation

```sh
npm install react-native-rn-as-modal
```

## Usage

```js
import RnAsModal from "react-native-rn-as-modal";

// ...

const result = await RnAsModal.multiply(3, 7);
```
